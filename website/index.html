Desktop HTML-to-video renderer

Turn motion code into clean video output.

CODE2VIDEO is a desktop app for rendering HTML, CSS, and JavaScript animations into MP4 files using Chromium and FFmpeg. It is built for demos, promo clips, UI motion, social videos, and product showcases.

Features

Built for code-driven motion.

Desktop rendering

The app runs locally instead of depending on a browser upload pipeline.

Real browser output

Frames are rendered through Chromium before FFmpeg handles the final encode.

Simple export flow

Preview, render, and save directly to the location you choose.
